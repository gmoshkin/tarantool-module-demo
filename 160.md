
# tlua

если код выполняется один раз, используем `eval`/`exec`
```rust
let lua = tarantool::lua_state();
lua.exec("
    box.cfg { listen = 3301 }
    box.schema.space.create('foo', ..)
    box.space.foo.create_index(..)
")?;

let memory: usize = lua.eval("return box.cfg.memtx_memory")?;
```
